# Ch 11: Tracking Your Weight and Adjusting Your Diet
Book: The Renaissance Diet 2.0 | Authors: Dr. Mike Israetel, Melissa Davis, Jen Case, James Hoffmann

## Core Principle
Track weight to assess progress. Make adjustments based on 2-3 week trends, never single weigh-ins.

---

## TRACKING BODYWEIGHT

### Protocol
- Weigh 2-3x per week (not daily)
- Same conditions: morning, after bathroom, before eating
- Compare **weekly averages**, not individual readings

### Why Weekly Averages
- Water weight masks tissue changes short-term
- Day-to-day fluctuations are noise
- Trends only visible across weeks

### Acceptable Fluctuation (Maintenance)
**±1.25% bodyweight** = normal range
- 150 lb person: 148-152 lb acceptable
- Check that weekly averages aren't trending up/down

---

## KEY FORMULA

**1 lb tissue ≈ 3,500 calories**

### To Calculate Adjustment
1. Find average weekly weight change (lbs)
2. Multiply by 3,500 = weekly calorie surplus/deficit
3. Divide by 7 = daily adjustment needed

**Example**: Losing 0.5 lb/week less than target
- 0.5 × 3,500 = 1,750 cal/week deficit needed
- 1,750 ÷ 7 = **250 cal/day to cut**

---

## ADJUSTING BY GOAL

### Maintenance
1. Run diet 3 weeks
2. If weight stable (±1.25%): continue
3. If trending up/down: calculate calorie difference, adjust accordingly
4. Reassess every 2-3 weeks

### Fat Loss
1. Set target: 0.5-1% BW/week
2. Run 3 weeks, track average weekly loss
3. If losing slower than target: cut more calories
4. If losing faster: can add back (optional)

**Which macros to cut (in order):**
1. Fats first (no unique benefit above minimum)
2. Carbs second (once fats at 0.3 g/lb)
3. Protein: never cut below minimum

### Muscle Gain
1. Set target: 0.25-0.5% BW/week
2. Run 3 weeks, track average weekly gain
3. If gaining slower: add calories
4. If gaining faster: can reduce (avoid excess fat)

**Which macros to add (in order):**
1. Carbs first (more anabolic)
2. Fats second (easier to eat, less anabolic)
3. Protein: rarely needed above 1 g/lb

---

## DIET PHASE TRANSITIONS

### Fat Loss → Maintenance
1. Calculate midpoint between current intake and estimated maintenance
2. Eat at midpoint for 2 weeks
3. If stable: increase 20% every 3-4 weeks until at full maintenance
4. If still losing: add calories to match loss rate
5. If gaining: hold current until stable

### Muscle Gain → Maintenance
1. Calculate weekly surplus from final 2 weeks of gain
2. Subtract that amount from daily intake
3. Run 3 weeks, adjust if weight not stable

### Fat Loss → Muscle Gain
1. Jump to estimated maintenance for current weight
2. Post-diet metabolism is suppressed → maintenance creates surplus
3. Run 3 weeks; adjust up if gain pace slows

### Muscle Gain → Mini-Cut
1. Use current intake as "maintenance"
2. Create deficit as normal (0.75-1% loss/week)

### Mini-Cut → Muscle Gain
1. Calculate maintenance + surplus based on new weight
2. Short mini-cut = minimal diet fatigue = accurate maintenance estimate

---

## COMMON MISTAKES

### Don't:
- Adjust based on single weigh-in
- Panic at week 1 or 2 fluctuations
- Weigh daily if it causes stress
- Ignore multi-week trends

### Do:
- Wait 2-3 weeks before adjusting
- Use weekly averages
- Account for hormonal cycles (females)
- Trust the math over single data points

---

## PRACTICAL TIPS

### Hormonal Bloat (Females)
- Weight can spike 2-5 lbs during certain cycle phases
- Compare same phase month-to-month
- Don't adjust diet based on hormonal week

### Post-Hard Training
- Inflammation = water retention
- Weight may be higher morning after hard session
- Account for this in weekly averages
